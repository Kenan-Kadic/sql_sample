WHAT DOES ALTER TABLE DO?
ALTER TABLE adds a new column to the table

example
 ALTER TABLE celebs
 ADD COLUMN twitter_handle TEXT;

HOW DO YOU CREATE A NEW TABLE?
You use the CREATE TABLE clause.

example
 CREATE TABLE celebs (
  id INTEGER,
  name TEXT,
  age INTEGER,
 );
